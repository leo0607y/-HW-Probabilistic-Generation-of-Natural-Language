# 自然言語の確率的生成の過程

## ▶目的
- 1バイト文字(半角文字：アルファベット＆数字＆記号)のみからなるテキストデータを用いて"自然言語の統計的性質"を調べる
## ▶準備
- ある程度の大量データが必須
- 特定のフォルダに著作権切れの電子化されたテキストのファイルを収取する
## ▶言語の種類
- 英語で特定の著作者の小説を使用することにする。
- 日本語は2バイト文字となるので第一段階では実験は行わない

## 実験の手順
1) 処理前の原文データをフォルダ(Unprocessed)に保存し、〇〇.txtのファイルを開きアルファベット(26種)＆空白＆改行コード(¥n)はそのまま、それ以外の記号などを空白に置換し、処理後は別フォルダに〇〇_processed.txtとして格納するコードの作成を行う。

2) 半角文字のみからなるテキストファイルから文字を1字ずつ読みこみ、ファイルに含まれる文字の出現率を調べる。大文字と小文字の区別を行った場合と行わなかった場合の両方の結果を表示する。表またはグラフを用いる。

3) 文字の二ッ組（連続する2文字）、三ッ組（連続する3文字）の出現率を調べる。各上位50位までを表またはグラフを用いて示す。

4) 2)で調べた文字の出現率に応じて任意の文字をランダムに100個ほど生成させる。乱数と確率計算を用いる。
- 簡便法
    1) テキストファイルの全文字数Mを調べる
    2) 乱数の原理によりM以下の任意の数kを取得する
    3) ファイル中のk番目の文字を調べて出力する
    4) 以上を繰り返す

5) 4)と同じ要領で二ッ組、三ッ組の出現率に応じてランダムに文字列を生成させる。以下は二ッ組の簡便法
    1) テキストファイルの全文字数Mを数える
    2) 乱数の原理によりM以下の任意の数ｋを得る
    3) ファイルの中でｋ番目の文字とK+1番目の文字を続けて出力する。ここでk+1番目の文字をAとする
    4) 再び乱数によりM以下の任意のk'を得る
    5) k'番目の文字より後で一番最初に出現するAを探す
    6) 見つかったらその次の文字を次のAとして出力し4)以下を繰り返す
